# web

Web API built with FastAPI framework.

## Directories

| Path | What | When to read |
|------|------|--------------|
| `routers/` | API route handlers | Adding/modifying endpoints |
| `static/` | Static assets (HTML, JS, CSS) | Frontend changes |
| `tests/` | Web API tests | Adding/modifying tests |

## Files

| File | What | When to read |
|------|------|--------------|
| `server.py` | FastAPI app setup, middleware | Server config |
| `dependencies.py` | FastAPI dependency injection | Shared dependencies |
| `models.py` | API request/response models | API schema |
