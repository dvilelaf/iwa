# contracts

Olas protocol contract wrappers.

## Directories

| Path | What | When to read |
|------|------|--------------|
| `abis/` | Contract ABI JSON files (generated) | Adding contract ABIs |

## Files

| File | What | When to read |
|------|------|--------------|
| `service.py` | ServiceRegistry contract wrapper | Registry interactions |
| `staking.py` | Staking contract wrapper | Staking interactions |
| `mech.py` | Mech contract wrapper | Mech interactions |
| `mech_marketplace.py` | Mech marketplace contract | Marketplace features |
| `activity_checker.py` | Activity checker contract | Activity verification |
| `base.py` | Base Olas contract class | Contract abstraction |
