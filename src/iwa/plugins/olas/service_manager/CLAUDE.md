# service_manager

Service lifecycle management for Olas autonomous services.

## Files

| File | What | When to read |
|------|------|--------------|
| `lifecycle.py` | Service deploy, update, terminate | Service lifecycle |
| `staking.py` | Stake, unstake, claim rewards | Staking operations |
| `mech.py` | Mech service management | Mech features |
| `drain.py` | Fund recovery from services | Fund management |
| `base.py` | Base manager class | Manager abstraction |
| `__init__.py` | Manager exports | Module API |
