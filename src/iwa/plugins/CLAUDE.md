# plugins

Protocol integrations loaded dynamically at runtime.

## Directories

| Path | What | When to read |
|------|------|--------------|
| `gnosis/` | Gnosis Chain plugins (Safe, CowSwap) | Gnosis protocol work |
| `olas/` | Olas plugins (Registry, Services, Staking) | Olas protocol work |

## Files

| File | What | When to read |
|------|------|--------------|
| `__init__.py` | Plugin registration | Adding new plugins |
