# gnosis

Gnosis Chain protocol integrations: Safe multisig and CowSwap DEX.

## Directories

| Path | What | When to read |
|------|------|--------------|
| `cow/` | CowSwap DEX integration | Swap features |
| `tests/` | Plugin tests | Adding/modifying tests |

## Files

| File | What | When to read |
|------|------|--------------|
| `plugin.py` | GnosisPlugin registration | Plugin setup |
| `safe.py` | Safe multisig operations | Safe wallet features |
| `cow_utils.py` | CowSwap utility functions | Swap helpers |
| `__init__.py` | Plugin exports | Module API |
