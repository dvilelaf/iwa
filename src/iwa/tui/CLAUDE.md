# tui

Terminal User Interface built with Textual framework.

## Directories

| Path | What | When to read |
|------|------|--------------|
| `screens/` | Main application screens | Screen navigation |
| `modals/` | Popup modal dialogs | Modal interactions |
| `widgets/` | Reusable UI components | Custom widgets |
| `tests/` | TUI tests | Adding/modifying tests |

## Files

| File | What | When to read |
|------|------|--------------|
| `app.py` | Main Textual application | App lifecycle, navigation |
| `workers.py` | Background workers | Async operations |
| `rpc.py` | RPC configuration UI | RPC settings |
| `__init__.py` | Module exports | Module API |
