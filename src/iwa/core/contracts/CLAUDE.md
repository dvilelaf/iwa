# contracts

Contract abstractions for interacting with on-chain contracts.

## Directories

| Path | What | When to read |
|------|------|--------------|
| `abis/` | Contract ABI JSON files (generated) | Adding contract ABIs |

## Files

| File | What | When to read |
|------|------|--------------|
| `contract.py` | Base contract class with call/transact helpers | Contract abstraction |
| `erc20.py` | ERC20 token contract wrapper | Token interactions |
| `multisend.py` | MultiSend batch transaction contract | Batch transactions |
| `__init__.py` | Contract exports | Module API |
