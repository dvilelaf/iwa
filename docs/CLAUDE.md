# docs

MkDocs documentation source files.

## Directories

| Path | What | When to read |
|------|------|--------------|
| `dev/` | Developer documentation | Contributing, releasing |
| `interfaces/` | Interface documentation | CLI, TUI, Web docs |
| `plugins/` | Plugin documentation | Gnosis, Olas docs |

## Files

| File | What | When to read |
|------|------|--------------|
| `index.md` | Documentation home page | Docs structure |
| `core.md` | Core module documentation | Core features docs |
| `security.md` | Security documentation | Security features docs |

## Commands

```bash
just docs-serve  # Serve docs locally
just docs-build  # Build static site
```
